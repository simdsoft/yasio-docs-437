    xxsocket Class¶

    Provides the functionality of low-level socket based on POSIX socket APIs, support std::move

    Syntax¶

    namespace yasio { namespace inet { class xxsocket; } }

    Members¶

    xxsocket::xxsocket Constructs a xxsocket object.
